# Keywork Skills

> Placeholder: This folder contains reusable skills that Keywork can invoke.

## Intended Contents

Skills are modular capabilities that can be loaded into Keywork's context. Examples:

- `git.md` - Git operations and conventions
- `testing.md` - How to write and run tests
- `refactoring.md` - Safe refactoring patterns

## Skill Format

Each skill should define:
- When to use it
- Required context
- Step-by-step instructions
